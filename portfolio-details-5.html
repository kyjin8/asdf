Project information

  • 프로젝트 소개: Bitcoin 계열인 Litecoin을 기반으로 해서 오라클 클라우드로 알트코인을 빌드하였습니다.
  • 핵심 기능: 코인 채굴, 코인 보내기/받기, 거래 정보 확인 등
  • 프로젝트 기간: 2021.6.7 ~ 2021.6.18
  • 제작 인원: 4명
  • WSL2, Ubuntu(20.04), Oracle Cloud(free-tier), Powershell
  • Project URL: 깃허브에서 소스 보기

기술 문서

Litecoin 0.15 버전 사용하여 빌드

코인 발행량 4200만 → 8400만개로 조정(amount.h)

코인 이름과 설명 수정 ⇒ 제네시스 블록 및 해쉬 수정(chainparams.cpp)

빠른 테스트를 위해 코인 가용 시작 개수 2로 수정(consensus.h)

자동채굴 가능한 batch 파일 만들어 powershell에서 사용 가능

conf 파일로 허용 rpc, tcp 포트 지정

채굴 난이도는 자동으로 조정하므로 난이도 변경 주기 수정(chainparams.cpp)
